### 安装
```
pod 'ZLNavigationBar', '~> 0.0.9'
```
--- 
### FAQ
- 搜索不到？
    - 终端执行
    ```rm ~/Library/Caches/CocoaPods/search_index.json```
    - 重新```pod search ZLNavigationBar```
